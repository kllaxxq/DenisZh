Преимущества

Ускорение разработки

Вместо создания отдельных классов для каждого элемента вы комбинируете готовые утилитарные классы прямо в HTML.

Гибкость и кастомизация

Добавление своих цветов, изменение отступов, кастомные шрифты.

Адаптивность без усилий

Достаточно добавить префикс к классу, чтобы задать стиль для определённого разрешения.
